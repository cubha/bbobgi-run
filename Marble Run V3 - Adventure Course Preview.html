Marble Run V3: Adventure Course

World: 2400 x 3200px | 뱀형 경로 | 카메라가 선두 추적

경로 흐름:
START(상단중앙) → 깔때기 낙하 → 우측 채널 이동 ↓ U턴 → 좌측 물레방아 상승 → 나선 하강 ↓ 핀존+시소 → 분기(고속/안전) ↓ 합류 → 최종 스프린트 → FINISH
3대 절대 규칙:
1. 끼임/이탈 불가 — 모든 경로 완전 밀폐
2. 영구 정지 불가 — 바닥 경사로 항상 진행
3. 역전 가능 — 충돌/지름길로 순위 변동

Sec 1: Start Funnel

넓은 깔때기 → 첫 채널로 진입. 밀폐

Sec 2: Right Traverse Channel

우측으로 이동하는 밀폐 파이프. 내부에 범퍼+핀으로 속도 차별화. 수평이동 밀폐

Sec 3: U-Turn Drop

반원형 U턴으로 방향 전환. 바깥쪽이 빠르지만 충돌 위험. 방향전환

Sec 4: Waterwheel Lift

핵심 구조물! 회전하는 물레방아가 구슬을 위로 끌어올림. 4개 날개가 회전하며 구슬을 스쿱 → 상단 출구로 배출. NEW 상승이동

Sec 5: Spiral Descent

나선형 하강 구간. 벽으로 둘러싸인 원형 통로를 따라 회전하며 내려감. 안쪽 라인 = 짧지만 충돌 多, 바깥 = 길지만 안전. 나선 밀폐

Sec 6: Obstacle Alley

좌측 이동 구간. 풍차 2개 + 시소 1개 + 트램폴린. 장애물에 튕겨 지름길 갭에 빠지면 순위 역전! NEW 역전포인트

Sec 7: Splitter + Dual Path

좌/우 분기. 좌: 급경사+핀(고위험고수익), 우: 완경사(안전). 운요소

Sec 8: Final Sprint

합류 후 가속 직선 → 마지막 풍차 관문 → 병목 → 결승! 클라이맥스

신규 세그먼트 타입

ChannelRamp 밀폐 경사 채널 (상하벽)
CurvedChannel 반원/호형 밀폐 채널
WheelLift 물레방아 상승장치
Trampoline 고탄성 바운스 면
ShortcutGap 지름길 구멍 (역전용)
