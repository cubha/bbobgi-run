## 5. 핵심 발견

### 완주 시간 대폭 단축
- **Before** (시간제한 30초): 항상 30초 + 결과 전환
- **After** (완주 기반): **~14초** — 1등 결정 즉시 슬로모 → 5초 후 종료
- 물리 엔진 기준 구슬이 트랙을 ~8~9초 만에 주파 (카운트다운 3초 + 슬로모 5초 포함 = ~14초)

### 게임 경험 변화
| 관점 | 시간제한 버전 | 완주 기반 버전 |
|------|-------------|-------------|
| 게임 길이 | 고정 30초 | **가변** (트랙 길이에 비례) |
| 긴장감 | 30초 타이머 바 | 완주 진행도 바 |
| 공정성 | 시간 내 미완주 = 순위 불확실 | **전원 완주 보장** (retire 제외) |
| 1등 결정 | 1등 완주 후에도 나머지 기다림 | 1등 즉시 → 슬로모 → 종료 |

### 권장 후속 조치
| # | 작업 | 우선순위 |
|---|------|---------|
| 1 | 트랙 길이 조정 — 현재 ~9초 주파는 너무 빠름. 세그먼트 추가 또는 마찰 증가 권장 | **높음** |
